Excercise 5 Get the classifier into your browser

  1. The predictive model markup language is a common export format for machine learning models. Use the R package pmml to export the classifier (if you want to go the easy way use a decision tree or a random forrest

  2.  Use the opencpu package to create a webserver that can return your model (tip use XML::saveXML to convert the returnvalue to a string)

  3. retrieve the XML from a webpage like this
     var pmml = new Promise((resolve,reject) => {
