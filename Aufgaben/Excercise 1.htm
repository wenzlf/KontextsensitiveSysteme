Excercise 2 Sensors

  1. Create a webpage that contains a form with a text field called (call it Context) and a toggle switch

    (Tip: https://www.w3schools.com/howto/howto_css_switch.asp)

  2. Add a EventListener to the DeviceMotionEvent that gets enabled on Toggle

    (Tip: https://www.html5rocks.com/en/tutorials/device/orientation/)

  3. Install a time series data base using docker (www.docker.com):

    docker run -p 8086:8086  
-v influxdb:/var/lib/influxdb    influxdb

    Create a new database called training

  4. Use influent (https://github.com/gobwas/influent) to write the sensor data to the database within the event listener

    (Tip: Use Batch and write asynchronously every second or so. You may already aggregate the results to 20 samples per seconds)

  5. Collect data for some context that may be inferred from accelerometer data

  6. Optional: Extend with more sensors (http://caniuse.com/ambient-light, https://developer.mozilla.org/en-US/docs/Web/API/Web_Audio_API/Visualizations_with_Web_Audio_API, https://developer.mozilla.org/en-US/docs/Web/API/Touch_events, )
